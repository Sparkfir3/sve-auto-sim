This is the code and scripting documentation for uSVE, an unofficial automated Shadowverse: Evolve simulator.

The GitHub repository is available [here][Repo].

Script files for cards are available in the [`SveScripts` folder][SVE Scripts Folder] in the repository. Learn how card scripting works in the [SVE script][SVE Script Page] section of the documentation.
